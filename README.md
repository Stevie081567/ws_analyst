************************************************************************************************************
Wallstreet trader Tools for desktop and mobile web applications
************************************************************************************************************
consists of
1.stock screener/ analyst for stock picking coming from sektor etf holdings list. Stock selections are then chosen through several key sektor rotation based value comparissons. 
Indicates picked for different trading methods (momentum, trend following, etc)

2. Market Sentiment Cockpit to evaluate sentiment of US stocks from Wall Street. Shows Fear&Greed and VIX index. Compares to SPY and several other indicators or values. Then recommends trading direction,i.e. long/short, bullish or bear sentiment.
Then shows which sectors are increasing and which are losing value -> to define best sectors for trading. Also shows save haven and raw materials indices/ key sector etfs.

4 files needed index.html, market_sentiment_dashboard.html, trading_scanner.html, manifest.json
Requires Alpaca Markets data or paper-trading account keys.
